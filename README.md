# MBBe

![image](https://media.forgecdn.net/attachments/description/1471806/description_f334d634-6909-4bb1-91cf-8d9d4d2967a9.png)

**MBBe** is an enhancement suite for **MinimapButtonButton**.  
It does **not** replace the original addon.

## Required Dependency

You must install **MinimapButtonButton** first:  
[https://www.curseforge.com/wow/addons/minimapbuttonbutton](https://www.curseforge.com/wow/addons/minimapbuttonbutton)

If MinimapButtonButton is missing, MBBe has nothing to enhance.

## What MBBe Does

*   Adds visual/UI enhancements on top of MinimapButtonButton
*   Provides a cleaner **circular icon** style option
*   Adds quick right-click access to MBBe options and MinimapButtonButton settings

## What MBBe Does Not Do

*   Does not provide standalone minimap-button collection
*   Does not duplicate MinimapButtonButton core functionality

## Installation

1.  Install **MinimapButtonButton** (required dependency)
2.  Install **MBBe**
3.  Reload UI (`/reload`)
